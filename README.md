# Camaly Digital Vision

Experiências digitais de alto impacto centradas em design, estratégia e performance.

## 🚀 Tecnologias

- **Vite** & **React** (SWC)
- **Tailwind CSS** (Estilização)
- **Framer Motion** (Animações imersivas)
- **Lucide React** (Ícones)
- **Radix UI** (Componentes acessíveis)

## 🛠️ Desenvolvimento

```bash
# Instalar dependências
npm install

# Rodar em desenvolvimento
npm run dev

# Gerar build de produção
npm run build
```
